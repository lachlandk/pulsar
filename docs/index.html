Pulsar

A data visualisation framework for the web.

API Reference →

No Unnecessary Boilerplate

Minimalist API intended to reduce the amount of code needed to write data visualisation web apps as much as possible.

Powered by HTML Canvas

Good performance means that webpages can handle many plots at once with animations while still maintaining an attractive framerate.

Familiar Syntax

Similar API to object-oriented interface of matplotlib, meaning that data scientists will understand user-generated code.
